# Zettelfuchs — Website

Statische Landing Page für **Zettelfuchs** (B2B-SaaS: Lieferscheine automatisch auslesen → Excel).
Eine Datei pro Seite, keine Build-Tools, Schriften lokal gehostet (kein Google-CDN).

- `index.html` — Startseite
- `impressum.html`, `datenschutz.html` — Rechtliches (§ 5 ECG / § 25 MedienG, DSGVO)
- `legal.css`, `fonts.css`, `fonts/` — Styles & lokale Schriften

**Hosting:** GitHub Pages → Domain `zettelfuchs.at` (siehe `CNAME`). Jeder Push auf `main` geht automatisch live.
